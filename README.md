# Nexus

Nexus is a personal workspace for learning notes, experiments, and project
work. The repository currently includes a lightweight Python scaffold that can
grow as the workspace takes shape.

## Python Environment

This project uses Python 3.12 and `uv` for dependency management.

Install dependencies:

```bash
uv sync
```

Run the current entry point:

```bash
uv run python main.py
```

## Dependencies

The base environment currently includes:

- `torch`
- `torchvision`
- `torchaudio`
